# ShopMind AI 🛍️

## 🚀 Overview

ShopMind AI is a smart retail shopping assistant designed to improve user shopping experience through intelligent recommendations and real-time cart optimization.

## 💡 Key Features

* AI-inspired product recommendation system
* Smart Budget Optimizer (suggests cheaper alternatives)
* Personalized suggestions using browsing history
* Interactive cart with dynamic price insights
* Clean, modern, responsive UI

## 🧠 Unique Selling Point (USP)

ShopMind AI doesn't just recommend products — it actively analyzes your cart and suggests smarter, cheaper alternatives in real-time, helping users save money while maintaining quality.

## ⚙️ How It Works

1. User enters a shopping query
2. System extracts keywords and category
3. Products are filtered and ranked
4. Cart stores items using localStorage
5. Optimizer suggests better-value swaps

## 🔐 Assumptions

* Static product dataset
* No real payment integration
* Fully frontend-based

## 🛠 Tech Stack

* HTML5, CSS3, JavaScript
* LocalStorage

## 📌 Future Scope

* Gemini API integration
* Live product data using Google APIs
* Payment gateway integration
